Target is Kismet Math Library

## Inputs

| Type | Name | Description |
| --- | --- | --- |
| vector | V1 | The first point. |
| vector | V2 | The second point. |

## Outputs

| Type | Name | Description |
| --- | --- | --- |
| real | Return Value | The distance between two points in the XY plane. |
