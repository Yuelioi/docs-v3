Returns true if a timer exists and is active for the given handle, false otherwise.

Target is Kismet System Library

## Inputs

| Type | Name | Description |
| --- | --- | --- |
| struct | Handle | The handle of the timer to check whether it is active. |

## Outputs

| Type | Name | Description |
| --- | --- | --- |
| boolean | Return Value | True if the timer exists and is active. |
