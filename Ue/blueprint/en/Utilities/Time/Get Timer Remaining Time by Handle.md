Target is Kismet System Library

## Inputs

| Type | Name | Description |
| --- | --- | --- |
| struct | Handle | The handle of the timer to time remaining of. |

## Outputs

| Type | Name | Description |
| --- | --- | --- |
| real | Return Value | How long is remaining in the current iteration of the timer. |
