Returns a substring from the string starting at the specified position

Target is Kismet String Library

## Inputs

| Type | Name | Description |
| --- | --- | --- |
| string | Source String | The string to get the substring from |
| integer | Start Index | The location in SourceString to use as the start of the substring |
| integer | Length | The length of the requested substring |

## Outputs

| Type | Name | Description |
| --- | --- | --- |
| string | Return Value | The requested substring |
