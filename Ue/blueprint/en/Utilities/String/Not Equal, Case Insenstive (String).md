Test if the input string are not equal (A != B), ignoring case differences

Target is Kismet String Library

## Inputs

| Type | Name | Description |
| --- | --- | --- |
| string | A | The string to compare against |
| string | B | The string to compare |

## Outputs

| Type | Name | Description |
| --- | --- | --- |
| boolean | Return Value | Returns true if the input strings are not equal, false if they are equal |
