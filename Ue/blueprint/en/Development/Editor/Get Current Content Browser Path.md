Attempts to get the path for the active content browser, returns false if there is no active content browser
or if it was a virtual path

Target is Editor Utility Library

## Outputs

| Type | Name | Description |
| --- | --- | --- |
| string | Out Path | The returned path if successfully found |
| boolean | Return Value | Whether a path was successfully returned |
