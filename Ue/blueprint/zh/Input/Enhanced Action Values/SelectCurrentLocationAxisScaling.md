Returns the current value of InputAction /VirtualCamera/Modifiers/AxisScalingModifier/Inputs/SelectCurrentLocationAxisScaling.SelectCurrentLocationAxisScaling. If input is disabled for the actor the value will be 0.

Note: If the value is being altered by an Input Trigger or Input Modifier (such as a Released trigger) then the value may be 0.
