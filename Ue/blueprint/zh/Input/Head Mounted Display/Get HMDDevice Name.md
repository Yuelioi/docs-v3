Returns the name of the device, so scripts can modify their behaviour appropriately

Target is Head Mounted Display Function Library

## Outputs

| Type | Name | Description |
| --- | --- | --- |
| name | Return Value | FName specific to the currently active HMD device type. "None" implies no device, "Unknown" implies a device with no description. |
