Computes the direction between two bones / sockets.

Target is Kismet Animation Library

## Inputs

| Type | Name | Description |
| --- | --- | --- |
| object | Component | The skeletal component to look for the sockets / bones within |
| name | Socket or Bone Name From | The name of the first socket / bone |
| name | Socket or Bone Name To | The name of the second socket / bone |
