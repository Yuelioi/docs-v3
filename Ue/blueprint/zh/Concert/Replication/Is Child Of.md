Checks whether ToTest is a child property of Parent.

Target is Concert Replication Blueprint Function Library

## Inputs

| Type | Name | Description |
| --- | --- | --- |
| struct | To Test | The property for which to check whether it is a child property |
| struct | Parent | The property which is supposed to be the parent of ToTest |

## Outputs

| Type | Name | Description |
| --- | --- | --- |
| boolean | Return Value | Whether ToTest is a child property of Parent |
